# Welcome to Project Label

**Project Label** displays your workspace name and VS Code profile right where you need them:

- **Status Bar** — always visible at the bottom
- **Window Title Bar** — visible in the OS title bar / taskbar
- **Sidebar Panel** — quick toggles and info in the Activity Bar

## Quick Start

1. Look at the **status bar** (bottom-left) — your project label is already there!
2. Click the **$(bracket-dot)** icon in the Activity Bar to open the sidebar panel
3. Run `Project Label: Open Settings Panel` from the Command Palette for the full visual editor

## Features at a Glance

| Feature | Description |
|---------|------------|
| Project Name | Auto-detected from workspace folder |
| Profile Name | Detected from VS Code profile |
| Custom Label | Override with any text |
| Window Title | Inject label into title bar |
| Native Title Bar | Linux support for visible titles |
| Copilot Silence | Mute all Copilot Chat sounds |

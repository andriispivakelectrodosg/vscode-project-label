# Customize Appearance

Make the label match your style.

## Icon

Use any [Codicon](https://microsoft.github.io/vscode-codicons/dist/codicon.html) icon:

| Icon Code | Preview |
|-----------|---------|
| `$(bracket-dot)` | Default â€” code-like |
| `$(folder)` | Folder icon |
| `$(account)` | Profile icon |
| `$(rocket)` | Rocket |
| `$(zap)` | Lightning bolt |
| *(empty)* | No icon |

## Color

Set with hex color code (e.g. `#b87333` for copper) or a theme color token.

Leave empty to use your theme's default status bar text color.

## Position

| Setting | Effect |
|---------|--------|
| **Alignment** | `left` or `right` side of status bar |
| **Priority** | Higher number = further left. Default `1000` |
